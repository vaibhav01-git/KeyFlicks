3. Open Chrome and navigate to `chrome://extensions/`
4. Enable **Developer mode** (toggle in top-right corner)
5. Click **Load unpacked** and select the extension folder

### What You Get

KeyFlicks replaces your default new tab page with a feature-rich interface that includes:
- Quick access keyboard shortcuts for popular developer sites
- Daily programming quotes for inspiration
- Enhanced Google search with glassmorphic design
- Beautiful cyberpunk hacker theme with Matrix effects
